# HAM 台网点名主控台使用说明

适用版本：V1.0.01

## 1. 软件用途

HAM 台网点名主控台用于业余无线电台网点名。主控可在一个界面中完成活动信息设置、友台呼号登记、监听源候选读取、主控发射提示、Excel / ADIF 日志导出、匿名使用统计设置和呼号资料辅助补全。

## 2. 快速开始

1. 在顶部填写台网活动名称、主控呼号、主控 QTH、主控设备、主控天线和主控功率。
2. 在右侧选择监听源，填写地址或通话组，点击“刷新”。
3. 从右侧最近通联列表单击友台，可加入左侧备选区；在备选区双击卡片可带入呼号输入区并移除该备选卡片。
4. 检查并补充天线、信号报告、备注等信息。
5. 点击“添加记录”写入主列表。
6. 活动结束后点击“Excel”或“ADIF”导出点名日志。

## 3. 顶部活动栏

- 台网活动名称：用于保存和导出时识别本次活动。
- 主控呼号：用于识别主控发射提示。
- 主控 QTH、设备、天线、功率：写入导出日志，便于活动留档。
- 已记录计数：点击可手动设置当前已记录数量，下一条序号自动顺延。
- 保存：保存当前点名表格。
- 自动保存：开启后在记录变更时自动保存。
- 新建：开始新的点名活动。
- 说明书图标：打开当前语言版本的使用说明。
- 语言图标：在中文和英文界面之间切换。

## 4. 添加记录

左侧录入区用于登记单个友台。

- 前缀：可单独输入呼号前缀数字。
- 呼号：必填，支持自动大写和呼号建议。
- QTH：填写地区或网格定位。
- 使用设备名称：填写电台、热点、车台或手台型号。
- 模式：如 FM、DMR、YSF 等。
- 功率：如 L、M、H、5W、25W。
- 信号报告：如 59 或 5/9。
- 天线：如车载、GP、八木、手台原装天线。
- 备注：记录热点、直频、首次参加、移动台等补充信息。

点击“添加记录”后，记录进入主列表。正在编辑已有记录时，按钮会变为“保存修改”。

### 4.1 备选区

备选区默认为空，用于临时放置需要稍后登记的友台。

- 右侧最近通联列表中置顶显示最近 6 个呼号，背景色与普通记录区分。
- 单击右侧候选行可加入备选区；备选区最多显示 4 张卡片，后加入的卡片在右侧。
- 双击备选区卡片可将呼号和资料带入录入区，并自动删除该备选卡片。
- 每张备选卡片右上角有 X，可单独删除；备选区右上角 X 可清空全部备选。
- 鼠标悬浮在备选卡片上会提示“双击卡片录入”。

## 5. 监听源使用

右侧“监听源”用于读取最近通联候选。

- FMO：填写 FMO 地址，并选择 ws 或 wss 协议。
- MMDVM：填写 MMDVM 主机地址，读取 Last Heard。本地 MMDVM 如同时具有 TS1、TS2 时隙，可在“时隙”中选择“全部 / TS1 / TS2”。选择 TS1 或 TS2 时只显示对应时隙的呼号。
- HAMBOX：填写 HAMBOX 地址，读取最近通联。
- BM DMR：填写 BrandMeister 通话组。
- 其他网络源：界面已预留，后续逐步接入。

开启“自动”后，软件会定时刷新候选。点击候选行可将该友台带入左侧录入区。

## 6. 主控发射提示

底部“主控发射”提示条会根据当前监听源判断主控呼号是否正在出现。该功能用于提醒主控当前发射状态，避免漏记或误判候选。

## 7. 已记录列表

主列表显示本次活动已登记的友台。

- 搜索框：按呼号、QTH、设备或备注过滤记录。
- 点击记录行：打开编辑窗口修改该条记录。
- 全选 / 取消：批量选择当前筛选结果。
- 删除选中：删除已勾选记录。
- Excel：导出当前点名日志。
- ADIF：导出 ADIF 格式日志，每条记录按友台呼号与主控呼号的一次通联保存，便于导入支持 ADIF 的日志软件。

## 8. 呼号资料库

呼号资料库用于辅助补全友台资料。

1. 勾选“启用呼号数据库”。
2. 如需共享同步，点击“注册”填写注册呼号、CRAC 操作证书号、常用 QTH 和常用服务器。
3. 提交审核后，导入作者提供的验证密钥。
4. 开启同步后，软件可更新共享呼号资料。

未注册时仍可使用本地已有的呼号建议和历史记录。

## 9. 保存与导出建议

- 活动开始前先确认活动名称和主控信息。
- 正式点名前建议开启自动保存。
- 长时间活动中可定期手动点击“保存”。
- 活动结束后导出 Excel，并检查序号、呼号和时间是否完整。
- 如需要和通用电台日志软件交换数据，可同时导出 ADIF。

## 10. 关于与匿名统计

- 页脚“关于”按钮可打开软件信息、项目链接和作者联系方式。
- 匿名使用统计默认开启，仅用于改善软件体验；当用户尝试关闭时，软件会提示“此统计仅用于改善软件体验”。

## 11. 常见操作

- 重新开始活动：点击“新建”。
- 修改序号起点：点击顶部“已记录”计数卡片。
- 修改已有记录：点击主列表中的记录行。
- 清空单个输入框：点击输入框右侧的 X。
- 查看说明书：点击右上角说明书图标。
- 切换英文界面：点击右上角语言图标。
